# **App Name**: CampusConnect

## Core Features:

- User Authentication: Secure authentication using college email ID and roll number as password.
- Dashboard Overview: Display student's results, attendance records, and fee dues in a clear and organized manner.
- History and Records: Ability for students to view a history of their results, attendance and fee dues.

## Style Guidelines:

- Primary color: College-branded color (e.g., blue or green) for a professional look.
- Secondary color: Light gray (#F5F5F5) for background and content separation.
- Accent: Teal (#008080) for interactive elements and highlights.
- Clean and modern layout with a focus on readability and user-friendliness.
- Use relevant icons to represent different sections (e.g., results, attendance, fees).
- Subtle animations and transitions for a smooth and engaging user experience.
